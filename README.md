# Quixical


Quixical is a qaballah quiz application.

## Install

- Go Native method
```bash
$ go get github.com/gnosthi/quixical
```
- WGET method
```bash
$ wget
```

## Usage
- "-h: display help"
- "-csv: specify the problems file in csv format (question,answer)"
